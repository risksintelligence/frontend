# RRIO Frontend

Next.js 14 dashboard for the RiskSX Resilience Intelligence Observatory, featuring GRII scorecards, regime probabilities, forecast panel, RAS widget, and anomaly feed.

## Development
```
npm install
npm run dev
```

### Environment
- `NEXT_PUBLIC_API_BASE_URL` pointing to the backend (e.g., `https://backend-1-671e.onrender.com/api/v1`).

### Build
```
npm run build && npm run start
```
